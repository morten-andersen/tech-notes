### Snap - Universal Linux Packages

[https://snapcraft.io](https://snapcraft.io/)

#### Search for a Package

```bash
snap find <package>
```

#### Information about a Package

```bash
snap info <package>
```

#### List installed Packages

```bash
snap list
```

#### Install a Package

```bash
snap install <package>
```

#### Remove a Package

```bash
snap remove <package>
```
